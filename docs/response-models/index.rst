===============
Response Models
===============

The purpose of this document is to introduce the two different response model
classes defined within the |ASB|. The target audience is developers who need to
understand how the |ASB|'s response models relate to `service description`_
`operations`_ and how these two classes can be extended to create powerful
applications with ease.
